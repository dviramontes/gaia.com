# gaia.com code-exercise API

## Endpoints:

### Longest Preview Media URL:
#### GET `/terms/:tid/longest-preview-media-url`

## Requirements:

- Node v7 or newer

	`nvm install v7`
	
	`nvm use v7`

## Running:

`npm install`

## Running:

`npm start`

Server runs on **localhost:4000**

## Development:

`npm run dev`
